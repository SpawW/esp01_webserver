# esp01_webserver

Sample code for a ESP01 with webserver and relay control

## Required hardware

- ESP01 board
- Relay module

## Features

- Control GPIO0 and Led Board 
- WiFi connection using HTTP and 2 control buttons

![Sample HTTP View](images/screenshot01.png)

## Default values

- LED Board and relay ON
